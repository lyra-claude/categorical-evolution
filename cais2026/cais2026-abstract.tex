\begin{abstract}
We observe that for any connected graph on $n$ vertices and $m$ edges,
the cycle rank is exactly $\beta_1 = m - n + 1$.
This is a textbook identity. It is also a methodological catastrophe:
every comparison of ``cyclic vs.\ acyclic'' communication topologies
in multi-agent AI systems simultaneously compares dense vs.\ sparse
communication, and no published benchmark separates the two.
At least ten deployed systems enforce acyclic (DAG) topologies
based on this confounded evidence. We escape the confound via
directed cycle count---which does vary independently of density---and
present the first controlled experiment (240 runs, $p < 10^{-7}$,
amplified $14\times$ on rugged landscapes). The result is a safety
problem: the topology of every DAG-based agent system is justified
by evidence that cannot distinguish the variable it claims to test.
\end{abstract}

\section{Definitions}
\label{sec:definitions}
%% ===================================================================

We require only standard graph theory.

\begin{definition}[Communication topology]
A \emph{communication topology} for $n$ agents is a directed graph
$D = (V, A)$ where $|V| = n$ and each arc $(u, v) \in A$ represents
a permitted message channel from agent $u$ to agent $v$.
\end{definition}

\begin{definition}[Density]
The \emph{density} of a topology $D = (V, A)$ is $\delta(D) = |A|/|V|$,
the mean out-degree.
\end{definition}

\begin{definition}[Cycle rank]
The \emph{cycle rank} (first Betti number) of the underlying undirected
graph $G$ of $D$ is $\beta_1(G) = |E| - |V| + c$, where $|E|$ is the
number of undirected edges and $c$ is the number of connected components.
For connected $G$: $\beta_1 = |E| - n + 1$.
\end{definition}

\begin{definition}[Directed cycle count]
The \emph{directed cycle count} $\kappa(D)$ is the number of distinct
simple (elementary) directed cycles in $D$~\citep{johnson1975finding}.
Unlike $\beta_1$, this quantity depends on the arrangement of arcs,
not just on $(n, |A|)$.
\end{definition}

\begin{definition}[DAG topology]
A topology $D$ is a \emph{DAG topology} if $\kappa(D) = 0$:
it contains no directed cycle of any length.
\end{definition}

\section{The Confound}
\label{sec:confound}
%% ===================================================================

\begin{theorem}[Cycle rank--density identity]
\label{thm:confound}
Let $G = (V, E)$ be a connected graph with $|V| = n$ and $|E| = m$.
Then:
\[
  \beta_1(G) \;=\; m - n + 1.
\]
\end{theorem}

\begin{proof}
A spanning tree of $G$ uses $n - 1$ edges. Each additional edge
closes exactly one independent cycle. The count of independent
cycles is therefore $m - (n - 1) = m - n + 1$. Equivalently, this
follows from the Euler characteristic of connected
1-complexes~\citep{hatcher2002algebraic}.
\end{proof}

This is not a correlation. It is an identity. For any fixed $n$,
cycle rank $\beta_1$ and edge count $m$ differ by the constant $n - 1$.

\begin{corollary}
\label{cor:confound}
For connected graphs on $n$ vertices, $\rho(\beta_1, Y) = \rho(m, Y)$
for any outcome variable $Y$. No experiment on undirected graphs can
attribute an observed effect to cycle structure rather than to density.
\end{corollary}

\paragraph{Why this is obvious.}
The identity $\beta_1 = m - n + 1$ appears in any algebraic topology
textbook~\citep{hatcher2002algebraic}. Every graph theorist knows it.
It requires no advanced mathematics. A reviewer's natural response is:
``Surely the multi-agent systems community accounts for this.''

\paragraph{Why it was missed.}
They do not. Table~\ref{tab:confound} lists ten recent multi-agent
systems---spanning ICML, AAAI, WWW, and NeurIPS venues---that enforce
DAG communication topologies. In every case, the evidence cited for
the DAG constraint compares topologies that simultaneously differ in
cycle structure \emph{and} edge density. Not one controls for density.
The confound is hidden not by mathematical difficulty but by
\textbf{disciplinary distance}: the teams building these systems
reason about graph structure without the algebraic topology in which
the identity lives. The result is that an elementary theorem
invalidates the evidence base for a design choice made in at least
ten production-track systems.

\section{Escaping the Confound}
\label{sec:escape}
%% ===================================================================

Theorem~\ref{thm:confound} applies to the undirected cycle rank
$\beta_1$. The directed cycle count $\kappa(D)$ (Definition~4)
escapes the confound: two directed graphs with identical $(n, |A|)$
can have vastly different $\kappa$ values, depending on how arcs are
arranged. To test whether cycle structure has any independent effect,
we hold density constant and vary $\kappa$.

\paragraph{Experimental design.}
We construct 8 directed graph families, all with $n = 8$ vertices and
$|A| = 16$ arcs ($\delta = 2.0$). The families span
$\kappa \in \{0, 0, 3, 10, 14, 20, 29, 47\}$---from pure DAGs to
highly cyclic structures---at \emph{identical density}. Each graph
serves as the migration topology for an island model genetic
algorithm (50 individuals per island, 400 total, 500 generations,
OneMax fitness). We run 30 independent seeds per topology
(240 runs total). By construction, any observed effect is
attributable to cycle structure alone.

\paragraph{Results.}
Directed cycle count has a significant, independent effect on
population diversity:
\begin{itemize}
  \item \textbf{ANOVA:} $F(7, 232) = 6.90$,
    $p = 1.85 \times 10^{-7}$ (generation~30)
  \item \textbf{Effect size:} $\eta^2 = 0.17$ (diversity),
    $\eta^2 = 0.24$ (fitness)
  \item \textbf{Direction:} $r = -0.68$ between $\kappa$ and
    mean diversity---more directed cycles accelerate convergence
  \item \textbf{Temporal profile:} effect peaks at generations
    20--50, then decays as all topologies converge on the
    unimodal landscape
\end{itemize}

\paragraph{Landscape amplification.}
On NK landscapes~\citep{kauffman1993origins} with tunable ruggedness,
the effect scales with problem difficulty: $\eta^2$ rises from
$0.05$ ($K{=}0$, smooth) to $\mathbf{0.69}$ ($K{=}6$,
rugged)---a $14\times$ amplification. Real-world agent tasks are
closer to $K{=}6$ than to a unimodal toy problem.

\section{Safety Implications}
\label{sec:safety}
%% ===================================================================

The confound creates three concrete safety risks:

\paragraph{1. Misattributed robustness.}
A DAG-based system may be reliable not because it lacks feedback
loops, but because it has fewer communication channels. If the system
is scaled by adding channels (for fault tolerance or richer
coordination), cycles will reintroduce themselves---with behavioral
consequences the designer has no evidence to predict, because the
evidence base never isolated the relevant variable.

\paragraph{2. Premature consensus.}
Our experiment shows that high-$\kappa$ topologies accelerate
convergence: agents reach agreement faster. In multi-agent
deliberation for high-stakes decisions, premature consensus is a
known failure mode~\citep{dochkina2026}. Topology controls the rate
at which this occurs, but no deployed system accounts for it.

\paragraph{3. The $17\times$ error amplification trap.}
Moran~\citep{moran2025error} reports that unstructured multi-agent
communication amplifies errors by approximately $17\times$. The
standard response is to impose DAG structure---but our theorem shows
this response conflates two interventions: removing cycles and
removing edges. Without density-controlled evidence, we cannot know
which intervention actually reduces errors.

\paragraph{Recommendations.}
\begin{enumerate}
  \item \textbf{Control for density.} Any topology comparison must
    hold edge count constant. Claims based on confounded comparisons
    should be treated as unvalidated.
  \item \textbf{Monitor diversity.} Cycle structure primarily affects
    behavioral diversity---the system's capacity to maintain multiple
    strategies. Task accuracy alone will not reveal topology effects.
  \item \textbf{Use directed cycle count.} The undirected $\beta_1$
    cannot separate topology from density. Directed $\kappa(D)$ can.
\end{enumerate}

%% ===================================================================
\section{Related Work}
\label{sec:related}
%% ===================================================================

Yang et al.~\citep{yang2025topology} identify topology as \emph{the}
underexplored dimension of multi-agent systems, proposing nine
research directions---none of which recognize the density--cycle
confound. HERA~\citep{hera2026} is the first system to track cycle
count during topology evolution, converging on ``compact chains with
strategically preserved cycles,'' but without density control. In
evolutionary computation, the topology--diversity relationship is well
established~\citep{tomassini2005spatially,skolicki2005influence}, but
all prior work uses undirected graphs and falls under the confound.
The directed cycle count as an independent experimental variable is,
to our knowledge, novel.

%% ===================================================================
\section{Conclusion}
\label{sec:conclusion}
%% ===================================================================

The cycle rank of a connected graph equals its edge count minus a
constant. This textbook identity means that every published
comparison of cyclic vs.\ acyclic multi-agent topologies is
confounded: it cannot distinguish ``removing cycles helps'' from
``removing edges helps.'' The identity is elementary. The fact that
ten systems at top venues missed it is not.

We escape the confound via directed cycle count and demonstrate a
significant independent effect ($p < 10^{-7}$), amplified $14\times$
on rugged landscapes. The safety implication is that the multi-agent
systems community is deploying topologies based on evidence that
proves nothing about the property it claims to test. Principled
topology design for safe AI must begin with experiments that actually
separate the variables.
